# Financial-Model
Excel-based financial model for a hypothetical firm, integrating projected Profit & Loss, Balance Sheet, and Cash Flow statements. The model applies growth-based forecasting, scenario assumptions, and supporting schedules to ensure internal accounting consistency.
This repository contains a self-initiated financial modelling project built in Microsoft Excel for a hypothetical company. The model integrates Profit & Loss, Balance Sheet, and Cash Flow statements, linked through consistent accounting relationships.

# Model Structure

Assumptions Sheet – Key growth rates, cost drivers, and scenario inputs used for forecasting
Income Statement (P&L) – Historical data and projected revenues, costs, EBITDA, and net profit
Balance Sheet – Forecasted assets, liabilities, and equity with accounting linkages
Cash Flow Statement – Indirect method cash flow derived from operating, investing, and financing activities
Supporting Schedules – Fixed assets, financial liabilities, and equity schedules to ensure model consistency
Checks & Balances – Internal validation to ensure the Balance Sheet balances across forecast periods

# How to Use the Model
1. Download the Excel file from this repository.
2. Update assumptions in the Assumptions sheet.
3. Review projected financial statements and cash flows.
4. Check balance validations in the control section.
